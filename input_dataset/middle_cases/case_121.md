# Тестовий кейс #121

```
Нова фіча: Quick Capture · дизайн і специфікація

Після воркшопу JTBD і 20 user interviews основний інсайт підтвердився:
Job to be done — "зафіксувати думку перш ніж вона зникне".

Специфікація Quick Capture:

Тригер: глобальний shortcut Cmd+Shift+Space (Mac) / Ctrl+Shift+Space (Win)
Дія: з'являється floating input поверх будь-якого вікна. 1 рядок тексту.
Submit: Enter або Escape. При Enter — думка іде в "Inbox" тайл.

Inbox тайл: автоматично створюється при реєстрації.
Юзер обробляє Inbox коли зручно — переміщує думки в потрібні тайли.

Чому це важливо: зараз треба відкрити браузер → знайти xTiles → відкрити тайл → писати.
Це 20-30 секунд. За цей час думка може зникнути.

Ризики:
→ Desktop app needed (shortcuts не працюють без інсталяції)
→ Desktop app — новий продукт. Scope збільшується.

Рішення: почати з Browser Extension як MVP. Потім desktop.
Тімлайн: browser extension до Q3 кінця.
```
