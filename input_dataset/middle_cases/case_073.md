# Тестовий кейс #73

```
Лист до команди · після закриття раунду B · шаблон

(Ще не написаний — готую шаблон на випадок хорошої новини)

Тема: Ми закрили раунд B. Що це означає для вас.

Дорога команда,

Сьогодні ми закрили раунд B на $Х млн. Це результат вашої роботи — 
не моєї, не Максима, а кожного з вас хто будував цей продукт щодня.

Що змінюється:
→ Ми наймаємо [X] нових людей в [Q]
→ Зарплати переглядаємо до ринкового рівня до [дата]
→ Офіс оновлюємо до [місяць]

Що НЕ змінюється:
→ Наша місія
→ Наші цінності
→ Формат роботи (hybrid залишається)

Попереду найцікавіший рік в нашій історії.

Дякую що вірили разом зі мною.
Ілона

P.S. П'ятниця — корпоратив. Деталі від Олі.
```
