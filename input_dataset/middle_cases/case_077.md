# Тестовий кейс #77

```
Нова ініціатива: xTiles for Education · рання розвідка

Несподіваний сигнал: три університети написали про корпоративний план.
Одна викладачка створила public тайл "Introduction to UX Research" — 4,000 переглядів.

Чи варто цілеспрямовано виходити на Education?

Потенціал:
→ 2.4M студентів університетів в Україні
→ PKM (Personal Knowledge Management) trend серед academic users
→ Low CAC: студенти поширюють органічно

Проблеми:
→ Education = low revenue (студенти платять мало або нічого)
→ Потрібен спеціальний plan і фічі (sharing, assignments, class management)
→ Відволікає від основного B2B руху

Моє рішення: не йти в Education активно зараз.
Але: безкоштовний plan для .edu accounts → органічний growth → майбутні B2B юзери.
Технічно: .edu verification → 1 тиждень роботи. Поставити в Q3 backlog.
```
