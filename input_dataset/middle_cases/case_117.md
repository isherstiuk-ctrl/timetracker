# Тестовий кейс #117

```
Менторська сесія з Олею Семенець · 3-й місяць

Оля зробила перший самостійний user interview — показала запис.

Що вона зробила добре:
→ Відкриті питання (не "чи подобається вам X?")
→ Пауза після відповіді — дала юзеру говорити
→ "Розкажіть детальніше" — 3 рази за 45 хвилин

Що ще вдосконалити:
→ Підводила до висновків ("тобто ви маєте на увазі що...") — небезпечно
→ Не записувала невербальне (коли юзер зупинився і замислився)
→ Наприкінці не запитала "чи є щось що я не запитала?"

Оцінка: 7/10. Для третього інтерв'ю в житті — дуже добре.

Домашнє завдання:
→ Прочитати "The Mom Test" Rob Fitzpatrick — до наступної сесії
→ Провести ще 2 інтерв'ю самостійно
→ Написати summary з трьох проведених інтерв'ю (синтез, не переказ)

Наступна сесія: 19 червня.
```
