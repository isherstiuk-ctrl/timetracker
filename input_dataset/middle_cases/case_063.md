# Тестовий кейс #63

```
Аналіз відтоку клієнтів Q1 · 5 churned Enterprise клієнтів

Втратили 5 Enterprise клієнтів в Q1. Загальна сума: $14,000 MRR.

Провела exit interviews з 4 з 5 (один не відповів).

Причини:

ТОВ "Техносвіт" ($4,200/міс): "Перейшли на Notion тому що там є SSO і ми вже
там зберігаємо внутрішні документи." → SSO
ФОП Кравченко ($1,800/міс): "Немає мобільного додатку. Наша команда в полях."
→ Mobile
Startup "Finova" ($3,500/міс): "Product Hunt-хайп пройшов, команда не використовує."
→ Adoption/Onboarding problem
Агенція "Креатив+" ($4,500/міс): "Знайшли Fibery — більше функцій для нашого процесу."
→ Features gap (competitor)

Спільний патерн: Enterprise churns через відсутність фіч (SSO, mobile),
не через продукт як такий.

Рекомендація: прискорити SSO і mobile. Саме те що борд підтвердив.
```
