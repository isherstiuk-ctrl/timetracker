# Тестовий кейс #62

```
Читацький клуб · квітень · рецензія "Атомних звичок"

Нашому клубу 2 роки. Зустрічаємось щомісяця вдома у когось із учасників.
Наступна зустріч: 30 квітня · 19:00 · у Тані.

Рецензія для обговорення (дедлайн 25 квітня):

Головне питання для клубу: "Кліар стверджує що система важливіша за мотивацію.
Чи погоджуєтесь? Де ця теорія не працює?"

Моя чесна оцінка (5/5 але не без критики):
+ Конкретні механізми (habit stacking, environment design)
+ Наукова основа але без зайвого жаргону
+ "1% better" — реально змінив моє мислення
- Трохи занадто оптимістично для людей з ADD або депресією
- Деякі приклади з американського контексту не резонують

Практика яку вже застосовую:
Телефон заряджається на кухні — 3 тижні. Результат: менше скролінгу вранці.

Запитання для дискусії: "Яка ваша найуспішніша і найнеуспішніша звичка?"
Підготувати 3-4 конкретних приклади щоб розговорити людей.
```
