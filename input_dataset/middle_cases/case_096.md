# Тестовий кейс #96

```
Кулінарна книга · особистий проект · неквапливо

Ідея що жила в голові рік: зібрати рецепти бабусі в красиву книгу.
Не для публікації — для сім'ї.

Бабуся Ганна — їй 78, живе в Полтаві. Готує з пам'яті, без записів.
Боюсь що рецепти загубляться.

Що є:
→ 12 рецептів що я записала в різний час (в нотатках телефону, безлад)
→ 3 фото бабусі за готуванням (2019-2021)
→ Старий зошит з 5-6 рецептами її руки

Що треба зробити:
→ Зателефонувати бабусі — записати ще 15-20 рецептів (треба 2-3 дзвінки по 1 год)
→ Відсканувати зошит
→ Зробити фото страв — наступний приїзд до Полтави (3 травня!)
→ Дизайн: мінімалістичний, з її рукописом на сторінках
→ Друк: 5 екземплярів для сім'ї

Дедлайн: до Різдва 2026. Без тиску — це про любов, не про дедлайни.
```
