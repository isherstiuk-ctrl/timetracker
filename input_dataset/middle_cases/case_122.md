# Тестовий кейс #122

```
Аналіз DAU/MAU · чому ratio не росте

DAU/MAU: 17.8% вже 3 місяці. Ціль 20%.

Що означає: якщо MAU = 47K, то щодня заходять ~8,400 юзерів.
Ідеал для productivity tool — 25-35% (Notion, Obsidian мають ~28%).

Гіпотеза 1: відсутність daily trigger
→ Ми не даємо юзеру причини заходити щодня
→ Notion відкривають бо там living документи
→ Slack відкривають бо там нові повідомлення
→ xTiles — заходять коли треба щось зберегти або переглянути

Гіпотеза 2: habit formation проблема
→ Більшість юзерів не мають "xTiles ritual"
→ Медитація, щоденник, планування — ці звички хтось будує, більшість ні

Гіпотеза 3: юзери бачать цінність але в "passive" режимі (зберіг і пішов)

Тест: запустити "Daily Digest" email (щодня о 08:00: "ваш контекст на сьогодні")
Персоналізований: нагадує про незавершені тайли, daily notes.
Очікування: +3-4% DAU/MAU за 30 днів.
```
