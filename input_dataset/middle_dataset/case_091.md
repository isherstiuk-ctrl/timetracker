# Список незавершених справ

Контракт з підрядником — підписати (висить 14 місяців, більше не можна).
Аналіз конкурентів — назначити на Марину, дедлайн 30 квітня.
Ревью перформансу — провести для всіх 4 членів команди до кінця кварталу.
Особисті фінанси — переглянути і оптимізувати (відкладаю 3 місяці).
Стоматолог — записатись, остання візит рік тому.
Рекомендаційний лист для Олега — він попросив, написати цього тижня.
Страховка — поновити до кінця місяця, інакше перерва в покритті.
