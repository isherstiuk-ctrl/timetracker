# MVP scope · портал для партнерів

Must have: авторизація партнерів, dashboard з метриками, завантаження документів.
Must have: API для інтеграції з нашою системою, базова аналітика.
Should have: кастомний branding для партнера, email нотифікації.
Could have: white-label варіант, розширена аналітика.
Won't have in MVP: мобайл додаток, AI features, multi-language.
Технічний стек: React (фронт), Node.js (бек), PostgreSQL, AWS.
Прототип: Ігор покаже на kick-off 17 квітня.
