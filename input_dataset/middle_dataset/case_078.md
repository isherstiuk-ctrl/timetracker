# Onboarding flow · нова версія

Проблема: 40% юзерів кидають на 3-му кроці, треба скоротити.
Новий flow: 4 кроки замість 7, ціль — "aha moment" якомога швидше.
Крок 1: реєстрація (email + пароль, без кредитки).
Крок 2: створити перший тайл — показати core value одразу.
Крок 3: запустити перший workflow — щоб юзер побачив результат.
Крок 4: запросити колегу (опціонально, не блокуємо якщо пропустять).
A/B тест: 20% нових юзерів на нову версію, міряти completion rate тиждень.
