# Checklist · кінець спринту

Всі задачі в Done або з поясненням чому не закрито.
Code review: немає відкритих PR старших 24 годин.
Деплой: всі зміни задеплоєні на production.
Тести: всі тести green, coverage не впала.
Документація: оновлена для всіх нових фіч і API змін.
Ретроспектива: проведена, action items записані в Jira.
Планування наступного спринту: заплановано на перший день.
