# Feedback від команди · анонімне опитування

Запустити анонімне опитування щоб зрозуміти настрій команди.
Питання: задоволеність роботою, процесами, комунікацією, менеджментом.
Інструмент: Google Forms або Typeform — налаштувати цього тижня.
Анонімність: важлива — люди мають відчувати що можуть говорити чесно.
Дедлайн для відповідей: тиждень після відправки.
Результати: обробити і поділитись з командою чесно (без цензури).
Action: на основі результатів скласти план покращень і звітувати по ньому.
