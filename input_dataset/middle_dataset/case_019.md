# Після демо для інвесторів

Реакція загалом позитивна — зацікавлені, але є питання по монетизації.
Хочуть детальніший фінансовий план на 3 роки — підготувати до наступної зустрічі.
Андрій Петренко надішле перелік питань на пошту протягом тижня.
Наступна зустріч: через 2 тижні, дату уточнять — стежити за листом.
Доопрацювати слайд з unit economics — зараз виглядає слабко.
Підготувати відповіді на питання які звучали сьогодні.
Якщо підуть на follow-up — підготувати term sheet до тієї зустрічі.
