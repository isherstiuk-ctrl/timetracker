# Ревью контракту · підрядник

Є дві задачі: переглянути контракт і узгодити умови — це одне і те саме.
Дедлайн був ще 2024-02-01 — вже більше року висить, треба закрити нарешті.
Ключові пункти для узгодження: строки, відповідальність, IP права.
Юрист переглянув — є 3 коментарі в Google Doc, треба відреагувати.
Підрядник чекає — написав вже двічі, більше не можна ігнорувати.
Погодити підписання до кінця цього тижня — або розірвати домовленості.
Після підписання: скан в Google Drive / Contracts / 2024.
