# Налаштування моніторингу

Datadog: є але налаштований мінімально — додати custom метрики.
Sentry: error tracking — є, але алерти не налаштовані на Slack.
Uptime monitoring: UptimeRobot безкоштовний план — апгрейдити для більш частих перевірок.
Алерти: налаштувати для: error rate >1%, latency >500ms, memory >80%.
Dashboard: Ігор робить загальний дашборд в Datadog — готово до п'ятниці.
Runbook: для кожного типу алерту написати що робити — Тетяна + Ігор.
On-call: ротація чергових на вихідних — домовитись з командою.
