# Remote work · guidelines для команди

Core hours: 10:00–16:00 всі онлайн і доступні — решта гнучко.
Комунікація: async first — спочатку Slack, потім дзвінок якщо складно.
Staus в Slack: оновлювати якщо недоступний більше 30 хвилин.
Документація: все важливе фіксувати в Notion, не тільки в Slack.
Зустрічі: тільки якщо треба — перевірити чи можна вирішити async.
Відгули: брати без зайвих питань, просто попередити команду.
Team нorms doc: Марина пише, буде готово до п'ятниці — всім переглянути.
