# Список для переговорів · підрядник

Ключові пункти: строки здачі, відповідальність за затримки, IP права.
Наш bottom line: не підписувати без чіткого SLA і штрафних санкцій.
Їх позиція: хочуть гнучкі строки, ми хочемо фіксовані.
Компроміс: фіксована дата + 2 тижні буфер з нашої сторони.
Юрист: Олексій Мороз переглянув, є коментарі в документі.
Не погоджуватись на: необмежені правки без доплати, відмова від SLA.
Якщо не домовимось: є два альтернативних підрядники в списку.
