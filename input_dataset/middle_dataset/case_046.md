# Щомісячний звіт · команда

Метрики продукту: DAU, MAU, retention, churn — зібрати з Amplitude.
Метрики бізнесу: MRR, нові клієнти, відтік, LTV/CAC — з CRM.
Метрики команди: velocity, bugs closed, deploy frequency.
Що вийшло добре цього місяця — 3–5 пунктів.
Що не вийшло — 3–5 пунктів, чесно і без прикрас.
Плани на наступний місяць — ключові задачі і очікувані результати.
Надіслати CEO до першого числа наступного місяця.
