# Нотатки зустрічі · design review

Макети готові на 80% — залишились екрани для мобайл версії.
Нові екрани здає дизайнер в п'ятницю — ревью запланувати на понеділок.
Зауваження по дашборду: кольори не відповідають гайдлайнам, виправити.
Зауваження по онбордингу: задовгий flow, скоротити з 7 до 4 кроків.
Позитивне: нова навігація всім сподобалась, залишаємо як є.
Дизайнер: Катерина Лисенко, k.lysenko@design.com.
Наступне ревью: понеділок після здачі мобайл екранів.
