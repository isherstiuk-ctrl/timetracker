# Roadmap комунікація · зовнішня

Публічний roadmap: показує загальний напрямок без дат — щоб не давати обіцянок.
Формат: Notion page або Trello board — юзери можуть коментувати і голосувати.
Оновлювати: щоквартально після planning — Марина відповідальна.
Що показувати: "In progress", "Planned", "Under consideration", "Not planned".
Що не показувати: конкретні дати, внутрішні метрики, competitor analysis.
Запустити: до презентації v3.0 — 29 квітня — це буде частиною анонсу.
Збирати фідбек: кнопка "Request a feature" на кожному пункті.
